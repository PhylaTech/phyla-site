Open source

Built in the open.

The tools we build for our own work, we tend to give away. Phyla maintains open-source software for the community we came up in, and contributes upstream to the projects our field relies on.

§   What we maintain, and where we contribute
nf-notion A Nextflow plugin that mirrors a pipeline run into Notion as a live-updating page, so an entire team can follow a genomics workflow in real time without anyone reading a log file. Nextflow pluginGroovy GitHub → dash-seqviz A Plotly Dash wrapper for the SeqViz DNA, RNA, and protein viewer, with annotations, primers, and restriction sites. Interactive sequence visualization without leaving the Python dashboards scientists already build in. Dash componentPython & JavaScript GitHub → mlflow-autogluon A community MLflow flavor for AutoGluon. It teaches MLflow to save, load, serve, and autolog AutoGluon's tabular, time-series, and multimodal predictors, so AutoML models drop straight into an existing MLflow tracking and deployment stack. MLflow integrationPython GitHub → Omni-Optimizer in pymoo A paper-faithful implementation of the Omni-Optimizer multi-objective algorithm, contributed upstream to pymoo, a widely used Python framework for multi-objective optimization. Adopted by the maintainer and now shipping in the library, with credit in the project's official documentation. Upstream contributionPython pymoo.org →

nf-notion and dash-seqviz are maintained under Full Spectrum Analytics, a co-founder's company, and are moving to Phyla. Peer-reviewed research from the team lives on the impact map.
